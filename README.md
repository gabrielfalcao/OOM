# OOM - One Or More

Lightweight Incremental Parser Generator inspired by Regular Expressions and [Parsing Expression Grammars](https://bford.info/pub/lang/peg/).


## Features


### Ready
- Simple Pure Rust API

### TODO

- Rust Macro DSL

  - https://github.com/tagua-vm/parser
